📥安装与准备工作（V2必读！）
下载文件：从本页面（GitHub Releases）下载最新的.zip压缩包。

解压：把下载的.zip文件解压，你会得到一个文件夹。

【重要！：点开这个文件夹，你会看到两个必须永远在一起的“黄金搭档”请一定确保它俩一直在一起，所以最好直接整个文件夹一起动。】分开的话使用exe文件会创建一个空白json文件，所以如果一开始使用没有关键词分类，可能就是json文件没有跟exe文件在一个文件夹里：

SimsModHelper_v2.exe（我们的主程序）

rules.json（你的专属分类大脑！所有的规则都存在这里！）

放置：把这个整个文件夹放到你喜欢的地方。
【强烈推荐：直接把这个文件夹放在你的.../The Sims 4/Mods文件夹里，方便统一管理！】

✨ 【保姆级教程】Sims-MOD-Helper V2.1 一分钟上手指南！
➡️步骤0：启动与设置（一次）

【！！！高能警告！！！】

请一定在每次使用exe文件前，关闭Windows实时防护！（或添加信任）我们的程序是安全的，但由于它是新软件，Windows可能会“误报”并删除它。 强烈建议：首次打开之前，把下载的压缩包备份一下，以便后续Windows误删后直接解压！

随后打开 SimsModHelper_v2.exe。

选择Mods文件夹：点击“MODS文件夹路径”旁的“浏览...”，选择你的Mods文件夹(默认文件夹位置:.C:\Users\【你的用户名】\Documents\Electronic Arts\The Sims 4\Mods)。

选择下载文件夹：点击“下载文件夹路径”旁边的“浏览...”，选择你电脑浏览器默认下载文件的位置(通常是下载或Downloads)，可以通过打开浏览器下载界面，选择一个下载好的文件，右键单击后左键选取查看文件所在位置进行确定。

该路径只需要设置一次，软件就会自动记住！

【注意，打开时如果底部按钮显示不全（类似扁扁的一条线），可以将光标放到界面上下边界上增加高度，然后按钮就正常显示了】

➡️V2灵魂功能：🧠自定义你的分类规则！
这是V2版本最强大的地方！
在程序顶部的菜单栏点击“设置” -> “编辑分类规则...” 。
一个弹窗会弹出，这就是你的“分类大脑”！
添加：点击“添加新规则”，比如关键词furniture，目标文件夹BB-建造模式。
编辑：选中一条规则（比如face mask-> SkinDetail-皮肤细节），点击“编辑”，把它改成Makeup-化妆品。
删除：选中您不需要的规则，点击“删除”。
保存：点击“保存并关闭”！
由此，你的规则库( rules.json)会被越用越顺手！

rules.json 就是你的“分类蓝图”。你在“目标文件夹”里写什么名字，程序就会自动创建和使用什么文件夹！
举个例子：
默认规则："hair": "Hair-头发"
程序会创建 Mods/Hair-头发 文件夹。
你的自定义规则：你打开编辑器，把规则改成 "hair": "MyPrettyHairs"
程序就会自动创建 Mods/MyPrettyHairs 文件夹！
你的嵌套规则：你甚至可以改成 "hair": "01_CAS/MyHairs"
程序照样会自动创建 Mods/01_CAS/MyHairs 这样的子文件夹！
结论： 你的 rules.json 完全决定了你的 Mods 文件夹结构。你拥有100%的自由，让软件完全按照你的习惯来整理！

【批量规则增减/高阶选项：你也可以直接右键单击rules.json文件，选择用记事本打开，进行批量规则修改，但是需要确保语法遵循前面的内容。】
【后续也可能在自己用一段时间后，发布比较好用的成品json文件，可以直接抄作业，也欢迎大家私信或者评论区讨论补充分享~】

➡️核心功能①：批量下载（流程）

【注意：重要！**在下载前，请尽量手动清理一下你的“下载”文件夹，如果有不在这次分类范围里的zip/rar/package或者任何有可能被判定为模组的文件，请一定清理，以免模组重复加入模组文件夹造成模组冲突！**尽管程序设置了只解压新下载的文件，但是测试时发现旧的下载文件如果不清理仍然会被重新移动进mods文件夹，该使用限制已经计划在后续更新中进行修复，但是目前需要遵循，**否则就需要在使用步骤2后在AA-待整理文件夹里手动整理了。**】

适用：场景你得到了一个长长的CC清单txt文件，想一口气下载了。

添加列表：点击“添加CC列表(.txt)”按钮，按住Ctrl或Shift键，一次性选中你的所有txt文件。

开始: 点击最大的【▶ 执行步骤①:打开所有下载网页】按钮！

手动下载：你的浏览器会自动打开所有网页，你只需要像个无情的“下载按钮点击器”一样，手动点完成所有下载。

➡️核心功能②：整理下载文件夹（可独立使用）

适用场景：

1. 你刚用完【功能①】下载了一堆文件。

2. 或者，你平时自己零散地在“下载”文件夹里下载了很多MODS。

使用方法：

直接点击【▶ 执行步骤②: 整理下载文件夹】！

智能判断：

如果您刚使用过步骤①，它只会处理新下载的文件。

如果你是直接运行，它会扫描整个下载文件夹，弹窗告诉你找到了多少个MODS文件，询问你是否要全部整理。

它的作用：它将所有MODS搬家到Mods/AA-待整理文件夹中，并自动解压缩！

【重要提示】：这一步完成后，建议去AA-待整理文件夹里检查一下。有模组的包里可能有好几个版本（比如高配版/低配版），作者会提示你只需要一个，另外你需要手动删除掉多余的。

➡️核心功能③：智能分类（可独立使用）

适用场景：

你刚用完【功能②】，AA-待整理文件夹里全是刚整理好的文件。

或者，你自己手动往AA-待整理文件夹里扔了一堆乱七八糟的MODS。

使用方法：

直接点击【▶ 执行步骤③: 智能分类“待整理”文件夹】！

见证奇迹：看着日志窗口里的文件“嗖嗖嗖”地飞到他们各自的分类文件夹里（Hair-头发, Lashes-睫毛...），超级解压！

自动创建：你没有某个分类文件夹（比如你没有BB-建造模式），别担心，如果程序会自动帮助创建！
